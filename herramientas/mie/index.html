Cómo funciona
este mapa

El MIE™ mapea las 7 relaciones que más impactan tus resultados estratégicos. Para cada una mides dónde está tu influencia hoy y dónde necesita estar — y defines el principio de influencia que trabajarás para cerrar esa brecha.

Piénsalo bien antes de comenzar: ¿qué 7 personas tienen el mayor impacto en tus resultados? Pueden ser tu equipo directo, pares, superiores, clientes clave o socios. No es un organigrama — es un mapa de influencia real.

Para cada relación indicarás: nombre y cargo, tu nivel de influencia actual (1-5), el nivel que necesitas (1-5), y el principio de influencia que mejor aplica para cerrar esa brecha.
